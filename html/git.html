Git

  • git init — создаёт новый пустой репозиторий Git в текущей папке.
  • git clone <ссылка> — копирует (клонирует) удалённый репозиторий на ваш компьютер.
  • git status — показывает текущее состояние репозитория: какие файлы изменены, добавлены в индекс или не отслеживаются.
  • git add <файл> или git add . — добавляет изменения в индекс (готовит их к коммиту). Точка означает «все изменённые файлы».
  • git commit -m "Сообщение коммита" — создаёт новый коммит с указанным описанием.
  • git log — показывает историю коммитов (журнал изменений).
  • git branch — отображает список веток; звёздочка (*) указывает на текущую ветку.
  • git branch <имя> — создаёт новую ветку.
  • git checkout <имя-ветки> или git switch <имя-ветки> — переключается на указанную ветку.
  • git push origin <имя-ветки> — отправляет локальные коммиты в удалённый репозиторий (обычно origin — это GitHub).
  • git pull — забирает изменения из удалённого репозитория и сразу объединяет (merge) их с текущей веткой.
